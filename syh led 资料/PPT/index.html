Execution Guide

SEO & SEM 执行指南

基于关键词差距 + 页面结构 + 内容选题三表对齐，四阶段执行。

⌵ 按 → 键逐阶段查看 ⌵ 0 / 4
阶段一 · 技术基础 (第 1–2 周)
  • 检查并清理所有测试页（lorem ipsum 等），301/noindex 处理
  • 统一 canonical 标签，确保每个主词只有 1 个目标页面
  • 提交优化后的 XML sitemap 到 Google Search Console
  • 修复产品页标题重复和摘要不匹配问题（panel、tube 等）
  • 检查内链结构，建立 Manufacturer / Outdoor / Commercial 三个集群
阶段二 · 核心页面重写 (第 2–4 周)
  • 首页重写：17+ years、ISO、月产能、sample time 放到首屏下方，强化 manufacturer 定位
  • Commercial / Industrial / Outdoor 三大分类页每页 800–1,200 词 + 应用场景 + 产品表 + FAQ
  • 产品子分类页统一模板：产品定义 + 参数 + 应用 + 定制 + FAQ + 询盘 CTA
  • 所有产品页增加 PDF spec / IES file / installation guide 下载 CTA
  • 每个分类页加 5–8 个 FAQ（MOQ、lead time、warranty、certification）
  • 产品详情页只做型号词，分类页做主词 — 清理重复和 canonical
阶段三 · 内容 + 场景页 (第 2–6 个月)
  • 每月 4–6 篇 B2B 内容，按 24 题选题日历执行（Buying / Manufacturer / Application / Comparison）
  • 新增 /applications/ 场景页：Parking Lot、Warehouse、Gas Station、Office、Retail
  • 新增独立 OEM/ODM 转化页：写流程、MOQ、图纸、打样、包装、质检
  • 每篇内容内链到 2–3 个产品分类页，形成集群权重
  • Quality Assurance 页面补 19-step testing、ISO、inspection report — 作为内链枢纽
  • 所有产品页加入 Get Quote / Request Sample / WhatsApp / Email CTA
阶段四 · SEM + 外链 (持续)
  • Google Ads 5 组精准匹配（Manufacturer / Supplier / High Bay / Outdoor / Commercial）
  • 否词排除 B2C 意图：amazon、walmart、near me、diy、christmas、grow light 等
  • B2B 外链来源：行业目录、采购指南、供应商列表、guest post
  • 每季度新增 20–30 外链域名，优先英文行业站点
  • 每月复盘 GSC 数据，按曝光词修正 Title 和 FAQ
  • 季度更新外链策略，目标 12 个月后 80–120 域名
